## Programma della Serata

### **Workshop: Introduzione a Git e controllo di versione**
**Serata - dalle 21 alle 23**

#### **Obiettivo**

Git è un sistema di controllo di versione distribuito open source che permette agli sviluppatori di tracciare e gestire le modifiche apportate ai file di un progetto nel tempo.

In questo incontro faremo un'introduzione allo strumento, parleremo di come poter lavorare in collaborazione con altri utenti e quali possano essere le vulnerabilità in termini di sicurezza informatica.

**Materiale necessario**: 
- Se puoi, porta con te un PC con Git installato
- Account GitHub/GitLab (opzionale, ma consigliato)

**Durata: 2 ore**

**Costo**: È richiesto un contributo minimo di **5 euro** per la partecipazione, questo ci aiuta a sostenere le attività dell'associazione. Per i soci la partecipazione è **gratuita**.
